# clase2

### 1. Encabezados

Se usan almohadillas o hashes (`#`). A más hashes, menor es el tamaño del título.

* `# Título 1`
* `## Título 2`
* `### Título 3`

### 2. Énfasis (Estilos de texto)

* **Negrita**: `**texto**` o `__texto__`
* *Cursiva*: `*texto*` o `_texto_`
* ~~Tachado~~: `~~texto~~`

### 3. Listas

* **Desordenadas**: Usa asteriscos, guiones o signos de suma (`*`, `-`, `+`).
* **Ordenadas**: Usa números seguidos de un punto (`1.`, `2.`).
* **Tareas**:
* `[ ] Tarea pendiente`
* `[x] Tarea lista`



### 4. Enlaces e Imágenes

* **Enlace**: `[Texto del enlace](https://www.ejemplo.com)`
* **Imagen**: `![Texto alternativo](url_de_la_imagen.jpg)`

### 5. Bloques de Código

* Para una palabra o línea corta: usa comillas simples invertidas: ``código``.
* Para **bloques completos**: usa tres comillas invertidas (```) al inicio y al final.

### 6. Otros elementos útiles

* **Citas**: Usa el signo de mayor que (`>`).
* **Línea divisoria**: Tres guiones medios (`---`).
* **Tablas**:

```text
| Columna A | Columna B |
| --------- | --------- |
| Celda 1   | Celda 2   |

```
